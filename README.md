# JSP 팀 프로젝트

## :white_check_mark: 개발 환경
- Java: OpenJDK 17
- Tomcat: 9.0.104
- MySQL: 8.0.33
- Maven: 최신
- IDE: IntelliJ Ultimate (Community는 JSP 미지원)

## :white_check_mark: 실행 방법

1. URL 복사:
https://github.com/iamjjh30/InternetProgramming


2. IntelliJ로 터미널에서 git clone https://github.com/iamjjh30/InternetProgramming.git 입력

3. `pom.xml` 자동 로딩 확인

4. Tomcat 설정:
- 아티팩트: `war exploded`
- context path: `/`
- 포트: `8090`
